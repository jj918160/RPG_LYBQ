话说天下之大势，分久必合，合久必分。公元280年，西晋取魏国而代之，
统一中原。
西晋末年，由于统治者不思进取，社会腐朽，导致了关外胡人对中原的觊觎。

304年冬天，氐族领袖李雄占成都，自称“成都王”；史称成汉，匈奴贵族刘
渊起兵于离石（今山西离石），史称汉赵。五胡乱华揭开序幕。

317年晋愍帝被杀，司马睿，在建康登基，是为晋元帝，历时五十二年的西晋
灭亡，东晋开始。

由于胡人的残暴杀戮，汉民族人口锐减，几近亡国灭种。汉人避难从黄河流域
大规模进入长江流域，
在长江下游江南建立东晋，从此神州大地一分为而，北国五胡十六国之间，
以及南国和北国之间，战乱不断，
420年六月，刘裕篡位称帝，是为宋武帝，建立南宋帝国，晋帝国灭亡。

东晋灭亡之后，南朝共经历四个朝代——宋、齐、梁、陈。梁是南朝第三
个朝代，建于公元502年，史称“南梁”，也称“萧梁”。
公元515年，北燕联合大渝，南楚攻梁，梁国三面受敌，危在旦夕，云南
穆王，皇六子萧景琰，以及赤焰军统帅林燮临危受命。

赤焰军以他们军用战术战技加上地势优劣与北燕20万皇属大军大战三天三夜
后，最终将其斩落马下。血战之后，精疲力尽时，却等来了同胞的屠杀。。。
